# Plan 01-03: CI/CD Baseline (Tests + Lint)

## Objective
Добавить базовый CI/CD: запуск pytest и ruff на каждый push/PR, защитить main и обеспечить видимые результаты проверок.

## Context Constraints
- Блокирующие проверки: pytest + ruff.
- main защищён: merge через PR + проверки.
- CI на каждый PR: полный набор тестов + линт.
- Автодеплой не нужен (ручной).
